## Thesis Template prepared for TeX training workshop at Malayalam Department of Kerala University

Install the following:

* texlive-xetex 
* texlive-latex-extra 
* texlive-lang-other
* fonts-smc-manjari 
* fonts-smc-uroob 
* fonts-sil-gentium 
* fonts-inconsolata 
* fonts-ubuntu

To compile:
`make`
